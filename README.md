# Flying-Polygon

Flying Polygon is a small Flappy Bird clone game made in Godot.

This project was mainly created to practice:

- Game logic
- Player movement
- Collision handling
- Basic UI in Godot

The gameplay is simple: control a polygon, avoid obstacles, and try to survive as long as possible.
